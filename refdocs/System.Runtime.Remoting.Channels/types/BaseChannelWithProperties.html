System.Runtime.Remoting.Channels.BaseChannelWithProperties Class

Assembly: Mscorlib.dll
Namespace: System.Runtime.Remoting.Channels
Summary
Provides a base implementation for channels that want to expose a dictionary interface to their properties.
C# Syntax:
public abstract class BaseChannelWithProperties : BaseChannelObjectWithProperties
Remarks
The BaseChannelWithProperties class handles the complexity of asking the sinks in a channel sink chain for their properties. After the channel constructs its sink chain (if applicable) it must set the BaseChannelWithProperties.SinksWithProperties property.

Notes to inheritors: When you inherit from BaseChannelWithProperties, you must implement the BaseChannelObjectWithProperties.Keys property and the BaseChannelObjectWithProperties.Item property.

Field: SinksWithProperties
Summary
Indicates the top channel sink in the channel sink stack.
C# Syntax:
protected IChannelSinkBase SinksWithProperties;
Remarks
Channel sinks provide a plug-in point that allows access to the underlying messages flowing through the channel as well as the stream used by the transport mechanism to send messages to a remote object. Channel sinks are linked together in a chain of channel sink providers, and all channel messages flow through this chain of sinks before the messages are serialized and transported.
