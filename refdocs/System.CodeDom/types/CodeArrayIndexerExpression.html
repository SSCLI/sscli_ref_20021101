System.CodeDom.CodeArrayIndexerExpression Class

Assembly: System.dll
Namespace: System.CodeDom
Summary
Represents an expression that indicates an array and a specific index or indices.
C# Syntax:
[Serializable]
public class CodeArrayIndexerExpression : CodeExpression
Remarks
CodeArrayIndexerExpression can be used to represent array indexers. Use CodeIndexerExpression for representing code (non-array) indexers.
Example
The following code creates a CodeArrayIndexerExpression named indexer that references the index represented by the value of an integer parameter named x, within the array named array1: 
          CodeArrayIndexerExpression indexer = new CodeArrayIndexerExpression( new CodeVariableReferenceExpression("array1"), new CodeArgumentReferenceExpression("x"));

This will generate the following code:

[C#]

array1[x]

[VB]

array1(x)
