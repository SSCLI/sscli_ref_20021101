System.Configuration.IgnoreSectionHandler Class

Assembly: System.dll
Namespace: System.Configuration
Summary
Provides a section handler definition for configuration sections read and handled by systems other than System.Configuration.
C# Syntax:
public class IgnoreSectionHandler : IConfigurationSectionHandler
Remarks
The configuration system completely parses configuration files to collect configuration information and throws exceptions when it encounters configuration sections that do not have a corresponding entry in the section.IgnoreSectionHandler provides a section handler for sections read by other systems to prevent configuration file parse exceptions. See the section handler definitions for the in the Machine.config configuration file for examples of sections that specify IgnoreSectionHandler as the section handler.
