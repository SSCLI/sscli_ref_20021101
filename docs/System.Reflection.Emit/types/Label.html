System.Reflection.Emit.Label Structure

Assembly: Mscorlib.dll

Summary
Represents a label in the instruction stream.Label is used in conjunction with the ILGenerator class.
C# Syntax:
[Serializable]
public struct Label
Thread Safety
Reflection Emit is thread-safe when using assemblies that were created with the AppDomain.DefineDynamicAssembly method with the Boolean parameter isSynchronized set to true.
