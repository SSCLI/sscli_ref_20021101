System.Runtime.Remoting.Channels.SinkProviderData Class

Assembly: Mscorlib.dll
Namespace: System.Runtime.Remoting.Channels
Summary
Stores sink provider data for sink providers.
C# Syntax:
public class SinkProviderData
Remarks
The sink provider data is a tree structure of configuration information stored in the configuration nodes. All sink providers must have a constructor that accepts an IDictionary of properties and an ICollection of these structures if they want to be used from a configuration file.
