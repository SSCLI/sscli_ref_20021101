Field: Infinite
Summary
A constant used to specify an infinite waiting period. This field is constant.
C# Syntax:
public const int Infinite;
Remarks
The value of this constant is -1. For threading methods that accept a timeout parameter, such as Thread.Sleep and Thread.Join, this value is used to suppress method behavior that would normally occur due to elapsed time.
